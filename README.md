# PansiyonUygulamasi
- Pansiyon Uygulaması, SQL Server veritabanı ve C# kullanılarak geliştirilmiş bir otomasyon projesidir.<br>
- Uygulamada Kullanıcı Girişi, Şifre Güncelleme, Müşteri Ekleme/Güncelleme/Silme/Mesaj, Odaların Durumu, Stok Takibi, Gazete ve Radyo özellikleri bulunmaktadır.
